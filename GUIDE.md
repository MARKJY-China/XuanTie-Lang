# 玄铁 (XuanTie) 语言指南 v0.3.0.x

玄铁是一门旨在融合“高熵信息含量”与“原生异步并发”特性的中文编程语言。它既能让 AI 更直观地理解逻辑因果，也为中文开发者提供了极高的表达效率。

## 1. 基础语法

### 变量与常量
```xuantie
变量 名字 = "玄铁"
常量 版本 = 0.3
```

### 打印输出
```xuantie
打印("你好，" & 名字)  // 使用 & 进行字符串拼接
```

## 2. 数据类型
- **基础**: `整数` (123), `小数` (3.14), `字符串` ("abc"), `逻辑` (真/假), `空` (空)
- **容器**: `数组` ([1, 2, 3]), `字典` ({"键": "值"})
- **进阶**: `结果` (Result), `任务` (Task)

### 索引访问
支持通过 `[]` 访问数组、字典和字符串的元素。
```xuantie
变量 文本 = "玄铁"
打印(文本[0]) // 输出: 玄

变量 列表 = [1, 2, 3]
打印(列表[1]) // 输出: 2

变量 映射 = {"名": "玄铁"}
打印(映射["名"]) // 输出: 玄铁
```

### 逻辑运算
支持标准的逻辑运算，并具备**短路求值**特性。
- `且` (AND): 两个条件都为真时结果为真。
- `或` (OR): 其中一个条件为真时结果为真。
- `非` (NOT): 对逻辑值取反。
- `-` (负号): 对数值取反。

```xuantie
如果 (1 > 0) 且 (2 > 1) { 打印("真") }
变量 结果 = 非 真 // 结果为假
```

## 3. 流程控制

### 条件判断
```xuantie
如果 10 > 5 {
    打印("正确")
} 否则 {
    打印("错误")
}
```

### 循环
```xuantie
变量 i = 0
当 i < 3 {
    打印(i)
    i = i + 1
}
```

## 4. 函数
```xuantie
变量 加法 = 函数(a, b) {
    返回 a + b
}
变量 结果 = 加法(1, 2)
```

## 5. 进阶特性 (v0.3.0+)

### 结果处理 (Result)
引入函数式错误处理，避免嵌套的 `if err != nil`。
```xuantie
变量 获取数据 = 函数() { 返回 成功("数据") }

获取数据()
    .接着(函数(内容) { 打印("成功: " & 内容) })
    .否则(函数(错误) { 打印("失败: " & 错误) })
```

### 异步与并行 (Async/Parallel)
原生支持并发，底层基于 Go 协程。
```xuantie
// 异步执行
变量 任务 = 异步 { 耗时任务() }
变量 结果 = 等待(任务)

// 并行执行多个块
变量 结果集 = 并行 {
    { 任务1() },
    { 任务2() }
}
```

### 异常捕获 (Try/Catch)
捕获运行时错误（如除零、未定义变量）。
```xuantie
尝试 {
    10 / 0
} 捕捉 (错误) {
    打印("捕获异常: " & 错误)
}
```

## 7. 内置标准库 (v0.3.1+)
玄铁开始构建其 IO 与网络能力。

### 文件 IO
- `文件.读取(路径)`: 返回 `结果`，包含文件字符串内容。
- `文件.写入(路径, 内容)`: 返回 `结果`，布尔值表示是否成功。

### 网络
- `网络.获取(URL)`: 返回 `结果`，包含网页正文内容。支持配合 `异步` 使用。

### 字符串处理
玄铁提供了强大的字符串处理内置函数。
- `字符串.长度(s)`: 获取字节长度。也可直接使用 `s.长度`。
- `字符串.子串(s, 起始, 长度)`: 截取子字符串。
- `字符串.替换(s, 旧, 新)`: 全量替换子串。
- `字符串.分割(s, 分隔符)`: 返回分割后的数组。

### 数学运算
- `数学.绝对值(n)`: 获取绝对值。
- `数学.正弦(n)`, `数学.余弦(n)`: 三角函数。
- `数学.平方根(n)`: 计算平方根。
- `数学.随机数()`: 返回 0.0-1.0 之间的浮点数。
- `数学.随机数(n)`: 返回 0-(n-1) 之间的整数。

### 时间处理
- `时间.现在()`: 获取当前 Unix 时间戳。
- `时间.休眠(ms)`: 程序暂停指定的毫秒数。
- `时间.格式化(ts, 格式)`: 将时间戳格式化为字符串。格式遵循 Go 的 layout (如 `2006-01-02 15:04:05`)。

### 模块管理
使用 `引用` 关键字加载其他玄铁文件。
```xuantie
变量 模块 = 引用 "lib.xt"
模块.某个函数()
```

## 8. 支持的转义符
- `\n`：换行
- `\t`：制表符
- `\r`：回车
- `\b`：退格
- `\f`：换页符
- `\"`：双引号
- `\\`：反斜杠
- 其他字符：直接输出
