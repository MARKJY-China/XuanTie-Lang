玄铁是一门旨在融合“高熵信息含量”与“原生异步并发”特性的中文编程语言。它既能让 AI 更直观地理解逻辑因果，也为中文开发者提供了极高的表达效率。

## 1. 基础语法
### 变量与常量
```java
设 名字 = "玄铁"
常 版本 = 0.13
```

### 字符串插值 (String Interpolation)
玄铁支持在字符串中使用 `#{表达式}` 进行变量或逻辑插值，这比使用 `&` 拼接更加简洁。

```java
设 人 = "开发者"
示("你好，#{人}！现在是 #{时.现()} 毫秒")
```

### 类型转换
玄铁提供内置函数进行类型强制转换，转换结果始终封装在 `结果` (Result) 中：

+ `整(x)`: 转换为整数。如果是字符串且格式不对，返回 `结果`。
+ `小数(x)`: 转换为小数。
+ `字(x)`: 转换为字符串。

**转换安全指南**:  
若转换失败（如 `整("abc")`），结果容器将标记为失败：

+ 直接访问 `.值` 将返回 `空` (Null)。
+ 推荐使用 `.或(默认值)` 或 `.解包()` 进行安全处理。

```java
设 s = "123"
设 n = 整(s).值 // 123
设 f = 小数(n).值 // 123.0
// ---
设 n = 整(s).解包() // 123，若失败则抛出异常
设 f = 小数("abc").或(0.0) // 0.0，安全降级
```

### 示出与输入
```java
示("你好，" & 名字)  // 使用 & 进行字符串拼接
设 回复 = 输("请输入：") // 获取用户输入
```

### 结果容器 (Result) 增强
`结果` 类型是玄铁错误处理的核心。除了 `.成功`、`.值`、`.错误` 属性外，**v0.12.0** 引入了完整的函数式组合能力：

+ `.接着(函)` (Then)：成功时执行，支持转换容器状态。
+ `.映射(函)` (Map)：成功时转换值，但不改变成功状态。
+ `.否则(函)` (Catch)：失败时执行。
+ `.复原(函)` (Recover)：失败时尝试恢复，将错误转为成功值。
+ `.执行(函)` (Finally)：无论成功失败均执行。
+ `.解包()` / `.断言()` (Unwrap)：成功则返回值，失败则立即抛出运行时错误。
+ `.或(默认值)` (GetOrElse)：成功则返回值，失败则返回提供的默认值。

```java
设 获取数字 = 函(s) { 返 整(s) }

// 链式处理示例
设 最终值 = 获取数字("123")
.映射(函(x) { 返 x * 2 })
.复原(函(e) { 返 0 })
.或(-1)

示(最终值) // 246
```

## 2. 数据类型
+ **基础**: `整` (123), `小数` (3.14), `字` ("abc"), `逻` (真/假), `空` (空)
+ **容器**: `数组` ([1, 2, 3]), `字典` ({"键": "值"})
+ **结果**: `结果` (Result) —— 用于错误处理，通过内置函数 `成功(值)` 或 `失败(错误)` 创建。
+ **进阶**: `任务` (Task), `字节` (Bytes)

### 类型推导与 FFI 增强 (v0.15.6)
玄铁进一步简化了类型系统的使用，并增强了底层交互能力。

+ **自动类型推导**: 声明变量时可省略类型注解，编译器将根据初始值自动推断。
+ **泛型语法支持**: 支持在 `型` 和 `函` 中定义泛型参数 `<T, U>`。
+ **外部函数接口 (FFI)**: 支持直接使用 `外 函` 声明并调用 C 语言原生函数。
+ **Arena 内存管理**: 针对短生命周期高频分配场景（如编译器），支持使用 Arena 内存池一键回收。

```java
// 1. 泛型类定义
型 盒子<T> { 
    设 内容: T
    函 造(物: T) { 此.内容 = 物 } 
}
设 盒 = 造 盒子<整>(123) 

// 2. 外部函数声明 (FFI)
外 函 printf(格式: 字, 参数: 整): 整
printf("你好, %lld\n", 42)

// 3. Arena 内存池使用
外 函 xt_arena_new(大小: 整): 整
外 函 xt_arena_use(池: 整)
外 函 xt_arena_destroy(池: 整)

设 池 = xt_arena_new(1024 * 1024)
xt_arena_use(池) // 后续所有分配均在池中进行
// ... 执行逻辑 ...
xt_arena_destroy(池) // 一键回收所有内存
```

### 模块化命名空间与文件引入 (v0.14.0)
玄铁支持模块化开发，允许通过 `引` 关键字引入外部文件。

#### 1. 显式命名空间引入 (推荐)
为了防止不同文件中出现同名的函数、变量或类产生**命名冲突**，玄铁设计了 `予` 关键字来隔离命名空间：

```java
// lib.xt
设 名字 = "玄铁库"

函 招呼(人) {
    示("你好，" & 人)
}

// main.xt
引 "lib.xt" 予 库
示(库.名字)
库.招呼("世界")
```

#### 2. 平铺引入 (自举阶段 LLVM 编译器行为)
在当前的**自举阶段**，为了加速底层编译器的成型，`引` 暂时退化为类似于 C/C++ 的 `#include` 行为。它会将目标文件解析后，将其内部的全局变量、函数和类定义直接“平铺”合并到当前的全局作用域中。

**注意**：这种方式不需要使用 `予` 关键字，但如果引入的多个文件包含同名的类或函数，会引发命名冲突（重定义错误）。

```java
// 临时平铺引入写法
引 "lib.xt"

示(名字)
招呼("世界")
```

### 类型注解 (v0.9.8)
玄铁支持**可选类型注解**，允许在声明变量、定义函数参数或指定函数返回值时，明确其预期的类型。

+ **基础注解**: `设 变量: 类型 = 值`
+ **联合类型 (Union)**: 使用 `|` 表示多种可能类型，如 `设 值: 字 | 整 = 123`。
+ **空安全 (Nullable)**: 使用 `?` 表示可为空，如 `设 名字: 字? = 空`。未加 `?` 的类型默认不允许赋值为 `空`。
+ **接口 (Interface)**: 使用 `口` 定义方法契约。类只要实现了接口定义的所有方法，即可被视为该接口类型。
+ **泛型 (Generics)**: 使用 `<T>` 定义泛型类或函数，支持类型参数化。支持使用 `承` 进行**泛型约束**，如 `<T 承 会飞的>`。
+ **自定义类**: 支持使用类名作为类型注解。

支持的内置类型：`整`, `小数`, `字`, `逻`, `数组`, `字典`, `字节`, `结果`, `空`。

```java
// 1. 泛型类
型 盒子<T> {
    设 值: T
    函 造(v: T) { 值 = v }
}
设 b: 盒子<整> = 造 盒子<整>(123)

// 2. 泛型函数
函 恒等<T>(x: T): T { 返 x } 

// 3. 泛型函数+约束
函 启动飞行<T 承 会飞的>(器物: T) {
}

// 4. 接口定义
口 会飞的 {
    函 飞行()
}
```

### 索引与属性访问
支持通过 `[]` 访问数组、字典和字符串的元素，支持通过 `.长度` 获取容器大小。

```java
设 文本 = "玄铁"
示(文本[0]) // 输出: 玄

示(文本.长度) // 输出: 2 (UTF-8 字节长度)

设 列表 = [1, 2, 3]
示(列表[1]) // 输出: 2
示(列表.长度) // 输出: 3
```

### 数组高级操作 (v0.12.0)
数组现在具备了工业级编程语言的成员方法：

+ `.追加(元素1, ...)`：在末尾添加元素。
+ `.截取(起始, [结束])`：截取子数组。
+ `.删(索引)`：删除指定位置元素。
+ `.插(索引, 元素)`：在指定位置插入元素。
+ `.包含(元素)` / `.包含?(元素)`：判断数组是否包含该元素。
+ `.空?` / `.非空?`：判断数组状态。
+ `.找(元素)`：返回元素第一次出现的索引。
+ `.映射(回调(x, i))`：返回一个经过回调函数处理后的新数组。
+ `.过滤(谓词(x, i))`：返回一个满足条件的元素组成的新数组。

```java
设 列表 = [1, 2, 3]
列表.追加(4).删(0) // [2, 3, 4]
设 结果 = 列表.过滤(函(x) { 返 x > 2 }) // [3, 4]
```

### 逻辑运算
支持标准的逻辑运算，并具备**短路求值**特性。

+ `且` (AND)
+ `或` (OR)
+ `非` (NOT)
+ `是` (IS)
+ `-` (负号)

```java
若 (1 > 0) 且 (2 > 1) { 示("真") }
若 1 是 整 { 示("1 是整数") }
```

## 3. 流程控制
### 条件判断 (若-抑-否)
```java
若 10 > 5 {
    示("正确")
} 抑 10 == 5 {
    示("等于")
} 否 {
    示("错误")
}
```

### 循环与遍历
**当 (While) 循环**
```java
设 i = 0
当 i < 3 {
    示(i)
    i = i + 1
}
```

**循 (Loop) 无限循环**
```java
设 k = 0
循 {
    k = k + 1
    若 k > 10 { 断 }
}
```

**遍历 (For-In) 循环**
```java
设 列表 = [10, 20, 30]
遍历 元素 于 列表 {
    示(元素)
}

遍历 元素, 索引 于 列表 {
    示("索引: " & 索引 & ", 值: " & 元素)
}
```

### 模式匹配 (v0.11.4)
```java
匹配 x {
    1 -> { 示("是数字 1") }
    "123" -> { 示("是字符串 123") }
    是 整 -> { 示("是任意整数") }
    _ -> { 示("默认分支") }
}
```

+ `型` (Type/Class)

### 运算符重载 (v0.13.0)
+ `_加_(另)`：对应 `+`
+ `_减_(另)`：对应 `-`
+ `_乘_(另)`：对应 `*`
+ `_除_(另)`：对应 `/`
+ `_等_(另)`：对应 `==`

```java
型 向量 {
    设 x; 设 y
    函 _加_(另) {
        返 造 向量(此.x + 另.x, 此.y + 另.y)
    }
}
```
