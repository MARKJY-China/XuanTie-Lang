# 09 — 玄铁渲染库编程指南 (v1.1)

> 底层基于 **raylib 6.0**，专注桌面 2D 渲染。本文档提供完整 API 参考、使用教程和实战示例。

---

## 1. 快速开始

```text
引 "../lib/渲染/渲染.xt" 予 R

R.初始化窗口(800, 600, "我的应用")

当 非 R.窗口应关闭() {
    R.开始绘图()
    R.清背景(30, 30, 50, 255)
    R.绘文字("Hello 玄铁", 100, 100, 20, 255, 255, 255, 255)
    R.结束绘图()
}

R.关闭窗口()
```

编译方式（Go 种子或自举编译器均支持渲染链接）：

```bash
# Go 种子
.\玄铁_go.exe tie .\程序.xt

# 自举编译器
.\玄铁.exe tie .\程序.xt
```

## 2. 窗口管理

### 2.1 创建与关闭

| API | 说明 |
|-----|------|
| `R.初始化窗口(宽, 高, "标题")` | 创建窗口，自动设置 60 FPS 目标帧率 |
| `R.窗口应关闭()` | 返回 `判`，用户点击关闭按钮或按 ESC 时为真 |
| `R.关闭窗口()` | 关闭窗口并释放 OpenGL 上下文 |
| `R.窗口就绪?()` | 窗口初始化完成返回真 |

### 2.2 属性控制

| API | 说明 |
|-----|------|
| `R.设置目标帧率(fps)` | 调整帧率上限（如 30、60、120） |
| `R.取帧率()` | 返回当前实际 FPS（整） |
| `R.取帧时间()` | 返回上帧耗时 秒（小数，double） |
| `R.取屏幕宽()` | 当前屏幕宽度（整） |
| `R.取屏幕高()` | 当前屏幕高度（整） |
| `R.设置窗口标题("新标题")` | 运行时修改窗口标题，**支持 Unicode（中文/日文等）** |

## 3. 绘图帧控制

每帧必须成对调用 `开始绘图` 和 `结束绘图`，所有绘制指令在二者之间。

| API | 说明 |
|-----|------|
| `R.开始绘图()` | 开始一帧，准备画布 |
| `R.结束绘图()` | 结束一帧，交换前后缓冲区 |
| `R.清背景(r, g, b, a)` | 用纯色填充背景，各通道 0–255 |

## 5. 文字渲染

### 5.1 默认字体（仅 ASCII）

| API | 说明 |
|-----|------|
| `R.绘文字("文本", x, y, 字号, r, g, b, a)` | 用 raylib 内置位图字体渲染（仅 ASCII 32–126） |
| `R.量文字("文本", 字号)` | 测量文本像素宽度（整） |
| `R.绘帧率(x, y)` | 在指定位置显示 FPS 计数器 |

### 5.2 自定义字体（v1.1 新增）

**支持格式**：`.ttf` / `.otf`（**不支持 `.ttc`**——raylib 限制）。

#### 加载模式

| API | 纹理大小 | 适用场景 |
|-----|---------|---------|
| `R.加载字体("路径.ttf", 字号)` | ~8192×4096（全 CJK） | 任意中文，不预知字符集。注意：字号 ≥28px 保证可读清晰度 |
| `R.加载字体精准("路径.ttf", 字号, "参考文本")` | ~256×256 | 已知渲染内容，追求极致清晰轻量 |

**全范围模式**内部加载 ASCII(32–126) + CJK 统一汉字基本区(U+4E00–U+9FFF, ~21K 码点)。纹理 ~32 MB。

**精准模式**从参考文本中自动解码 UTF-8 提取所有非 ASCII 码点（去重），仅加载 ASCII + 这些码点。纹理 ~128 KB，100% 不缺字。

> 建议：开发阶段用精准模式（快、清晰），发布前确实需要全 CJK 范围再切换到全范围模式。

#### 使用

| API | 说明 |
|-----|------|
| `R.绘字体文字(句柄, "文本", x, y, 字号, 间距, r, g, b, a)` | 用指定字体渲染（间距通常 1–2） |
| `R.释字体(句柄)` | 释放字体 GPU 资源 |

**完整示例**：

```text
设 显示文本 = "玄铁3D球体旋转"

// 精准模式：参考文本确保所有目标字符被加载
设 字体 = R.加载字体精准(
    "assets/HarmonyOS_Sans_SC_Regular.ttf",
    28,
    显示文本 & "Lambertian 光照 FPS0123456789|"
)
设 字体小 = R.加载字体精准(
    "assets/HarmonyOS_Sans_SC_Regular.ttf",
    18,
    显示文本 & "Lambertian 光照 FPS0123456789|"
)

当 非 R.窗口应关闭() {
    R.开始绘图()
    R.清背景(15, 15, 35, 255)
    R.绘字体文字(字体, 显示文本, 20, 20, 28, 2, 255, 255, 255, 255)
    R.绘字体文字(字体小, "Lambertian 光照 | 60 FPS", 20, 54, 18, 1, 200, 200, 220, 255)
    R.绘帧率(700, 20)
    R.结束绘图()
}

R.释字体(字体)
R.释字体(字体小)
```

## 6. 纹理与图像

加载 PNG 等格式到 GPU 纹理，可高效绘制。

| API | 说明 |
|-----|------|
| `R.加载纹理("路径.png")` | 从文件加载纹理，返回整型句柄 |
| `R.释纹理(句柄)` | 释放 GPU 纹理资源 |
| `R.绘纹理(句柄, x, y, tr, tg, tb, ta)` | 原尺寸绘制（t 参数为着色，255/255/255/255 = 无着色） |
| `R.绘纹理Ex(句柄, x, y, 旋转角, 缩放, tr, tg, tb, ta)` | 带旋转（度）和缩放倍率的绘制 |

## 11. 完整示例：旋转球体 Demo

参见项目源码：[examples/旋转球体.xt](../examples/旋转球体.xt)

```text
引 "../lib/渲染/渲染.xt" 予 R

R.初始化窗口(800, 600, "XuanTie Render — Rotating Sphere")

设 字体 = R.加载字体精准(
    "assets/HarmonyOS_Sans_SC_Regular.ttf",
    28,
    "玄铁3D球体旋转"
)

设 角度 = 0
设 球X = 400
设 球Y = 300
设 球R = 140

当 非 R.窗口应关闭() {
    角度 = 角度 + 1
    若 角度 > 360 { 角度 = 0 }

    R.开始绘图()
    R.清背景(15, 15, 35, 255)

    // 网格背景
    设 gx = 0
    当 gx < 800 { R.绘直线(gx, 0, gx, 600, 25, 25, 50, 60) ; gx = gx + 40 }
    设 gy = 0
    当 gy < 600 { R.绘直线(0, gy, 800, gy, 25, 25, 50, 60) ; gy = gy + 40 }

    // 球体主体——多层同心圆模拟光照
    设 层 = 0
    当 层 < 20 {
        设 t = 层 * 1.0 / 20.0
        设 层R = 球R - 层 * 7

        设 nY = 1.0 - t
        设 亮度 = 0.15 + 0.85 * nY
        若 亮度 < 0.15 { 亮度 = 0.15 }
        若 亮度 > 1.0 { 亮度 = 1.0 }

        设 rr = 40 * 亮度
        若 rr > 255 { rr = 255 }
        设 gg = 80 * 亮度
        若 gg > 255 { gg = 255 }
        设 bb = 180 * 亮度
        若 bb > 255 { bb = 255 }

        R.绘圆(球X - 5*t, 球Y - 8*t, 层R, rr, gg, bb, 255)
        层 = 层 + 1
    }

    // 高光
    设 高光偏X = 球R * 0.45 * (角度 - 180) / 180.0
    设 hx = 球X + 高光偏X
    设 hy = 球Y - 球R * 0.35
    R.绘圆(hx, hy, 16, 255, 255, 255, 100)
    R.绘圆(hx - 2, hy - 2, 10, 255, 255, 255, 160)
    R.绘圆(hx - 3, hy - 3, 5, 255, 255, 255, 240)

    R.绘字体文字(字体, "玄铁3D球体旋转", 20, 20, 28, 2, 255, 255, 255, 255)
    R.绘帧率(700, 20)

    R.结束绘图()
}

R.释字体(字体)
R.关闭窗口()
```

## 12. 故障排查

| 现象 | 原因 | 解决 |
|------|------|------|
| 中文显示 `???` | 码点不在加载范围内 | 用 `加载字体精准` 并在参考文本中包含所有目标字符 |
| 字体加载但全部 `???` | `.ttc` 文件 | 换用 `.ttf` / `.otf` |
| 字体模糊 | 全范围模式下字号太小 | 字号提升到 28px 以上，或切换到精准模式 |
| `invalid redefinition` | 自举编译器旧版本 | 更新到 v0.18.6+ |
| 链接 `XT_InitWindow` 未定义 | 自举编译器未启用渲染链接 | 确保 `render/渲染桥.c` 和 `libraylib.a` 存在 |
| 窗口关闭按钮无效 | 自举编译器旧版本 | 更新到 v0.18.6+ |
