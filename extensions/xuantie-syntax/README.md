# 玄铁编程语言VSCode扩展

玄铁 (XuanTie) 编程语言的官方 VS Code 扩展包。
当前支持版本: v0.18.0

## 功能特性

- **语法高亮**：支持玄铁语言的所有关键字、运算符、字符串插值（#{}）及内置函数。
- **代码片段**：支持代码片段补全，如变量、函数、循环等（此功能需要在扩展设置中开启）。
- **智能补全**：扩展支持自动检索解析当前玄铁代码和引用的代码所声明的变量、函数、型等快速补全。
- **大纲导航**：在代码编辑器中显示当前文件的大纲导航，方便快速跳转到代码的指定位置（即Ctrl+Shift+O开启）。
- **代码跳转**：支持从代码中跳转到定义位置（如Ctrl+点击变量名、函数名、型名等）。

## 使用方法

安装此扩展后，所有以 `.xt` 结尾的文件将自动开启语法高亮、代码片段补全、智能补全、大纲导航、代码跳转等功能。
